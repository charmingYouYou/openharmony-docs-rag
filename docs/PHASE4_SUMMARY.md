## 已知问题

### 1. Mermaid 图表语法错误
**问题**：系统架构图因为 Mermaid 语法错误无法在飞书中渲染

**错误信息**：
```
Lexical error on line 10. Unrecognized text.
...        Query[/query]        Retrieve[/
```

**原因**：飞书的 Mermaid 解析器对某些语法支持有限，特别是节点标签中的特殊字符（如 `/`）

**解决方案**：
- 当前：图表已降级为代码块，用户可以查看源代码
- 未来：可以在飞书中使用飞书画板手动绘制架构图

### 2. 图表渲染限制
**问题**：飞书的 Mermaid 渲染器可能不支持所有 Mermaid 特性

**影响**：部分复杂图表可能无法完美渲染

**解决方案**：
- 简化图表语法
- 使用飞书画板手动绘制
- 保持本地 Markdown 文档作为参考
